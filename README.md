# AI Personal Assistant Automation (n8n)

This project is an AI-powered personal assistant built using n8n workflow automation.

## Features
- Telegram AI assistant
- Email automation
- Contact management
- Calendar scheduling
- Voice message transcription
- AI content generation

## Technologies
- n8n
- Groq LLM
- Google Gemini API
- Telegram Bot API
- Gmail API
- Google Calendar API
- Google Sheets API

## Workflow

Import the JSON workflow into n8n to run the project.
